# Employee Software Request Portal (ServiceNow Project)

## Overview

This project demonstrates a system developed on the ServiceNow platform where employees can request licensed software installations through the Service Catalog. The system helps manage and track software requests in an organized way.

## Features

* Software request through Service Catalog
* Approval workflow for installation requests
* Organized documentation and screenshots of the project

## Project Structure

* **Documentation** – Contains the detailed project documentation.
* **Project Screenshots** – Shows screenshots of the ServiceNow implementation.
* **LICENSE** – Contains project license information.

## Conclusion

This project shows how ServiceNow can be used to manage software installation requests efficiently and improve IT service management processes.
